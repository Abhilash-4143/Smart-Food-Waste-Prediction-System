🍽️ Smart Food Waste Prediction System

This project is an end-to-end Machine Learning solution designed to reduce food waste by predicting the number of meals required in large-scale food service environments such as restaurants, cafeterias, and catering services.

🚀 Overview  
Food preparation is often based on manual estimation, leading to overproduction or shortages. This system uses historical data and real-world factors like expected customers, weather conditions, festivals, and past consumption trends to make accurate predictions.

🎯 Key Features  
- Complete ML pipeline (data preprocessing, training, evaluation, deployment)  
- Feature engineering using a Weekend indicator  
- Exploratory Data Analysis (EDA) for pattern understanding  
- Model evaluation using MAE and RMSE  
- Flask-based web application for real-time predictions  
- Clean and modern user interface  

🛠️ Tech Stack  
Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn, Flask, HTML, CSS  

⚙️ How It Works  
Users input relevant details such as day, weather, expected customers, and previous consumption. The Flask backend processes this data and uses a trained Random Forest model to predict the number of meals to prepare.

📊 Impact  
This system helps reduce food waste, optimize resource usage, and support data-driven decision-making in food service operations.
